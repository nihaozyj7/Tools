# 工具脚本 & 常用软件合集

这是一个集合了我日常使用的小工具的项目，旨在提高工作效率和解决常见任务问题。

## 项目概览

本项目包含了多个实用工具，每个工具都专注于解决特定的问题或完成特定的任务。

## 工具列表

### 1. 大文件AI翻译
- **功能**: 专门用于处理大文件的AI翻译工具，能够将大文件分块处理并通过AI进行翻译
- **适用场景**: 文档翻译、技术文档本地化、大型文本文件处理

### 2. 歌词微调
- **功能**: 用于对歌词进行微调和优化的工具
- **适用场景**: 音乐创作、歌词编辑、时间轴调整

### 3. 图片压缩工具
- **功能**: 提供图片压缩功能，减少文件大小同时保持图片质量
- **适用场景**: 网站优化、文件传输、存储空间优化

### 4. 音视频响度归一化工具
- **功能**: 对音频和视频文件进行响度归一化处理，使播放音量保持一致
- **适用场景**: 音乐播放列表、视频制作、音频后期处理

## 使用说明

每个工具都有独立的README文件，详细介绍了工具的功能、使用方法和注意事项。请进入相应目录查看具体使用说明。

## 环境要求

- Python 3.x (针对Python脚本工具)
- 相关依赖库 (详见各工具README)
- 根据不同工具可能需要特定的API密钥或配置

## 注意事项

- 在使用各工具前，请确保已安装所需依赖
- 部分工具可能需要网络连接
- 大文件处理可能需要较长时间，请耐心等待
- 使用AI相关工具时请注意相关服务的使用条款和隐私政策
