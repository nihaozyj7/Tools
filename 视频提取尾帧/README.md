# 视频提取尾帧工具

## 功能
- 从MP4视频文件中提取最后一帧图像
- 支持批量处理多个视频文件
- 提供图形化界面，操作简单直观

## 适用场景
- 视频预览图生成
- 封面制作
- 视频内容快速查看
- 视频结尾帧提取

## 使用方法

### 环境要求
- Python 3.x
- OpenCV (`pip install opencv-python`)
- PyQt6 (`pip install PyQt6`)

### 安装依赖
```bash
pip install opencv-python PyQt6
```

### 运行工具
```bash
python 视频提取尾帧.py
```

### 操作步骤
1. 运行脚本后会打开图形界面
2. 点击"浏览"按钮选择要处理的MP4文件
3. （可选）点击"浏览"按钮选择输出目录，如不选择则默认保存在原视频同级目录
4. 点击"提取尾帧"按钮开始处理
5. 处理完成后会弹出提示框，显示保存路径

## 输出结果
- 提取的尾帧会保存为PNG格式图片
- 文件名格式为：`{原视频文件名}_last_frame.png`
- 例如：原视频为`example.mp4`，则输出为`example_last_frame.png`

## 注意事项
- 仅支持MP4格式视频文件
- 确保视频文件完整且未损坏
- 处理大视频文件可能需要一些时间
- 如遇到错误，请检查视频文件格式和完整性

## 依赖库说明
- `cv2` (OpenCV): 用于视频处理和帧提取
- `PyQt6`: 用于创建图形用户界面
